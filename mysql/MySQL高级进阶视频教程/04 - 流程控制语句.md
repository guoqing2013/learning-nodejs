# 03 - 流程控制语句

## IF语句

```
IF search_confition THEN statement_list
ELSEIF search_confition THEN statement_list
ELSE statement_List
END IF
```

##  CASE语句

```
CASE case_value
WHEN when_value THEN statement_list
WHEN when_value THEN statement_list
ELSE statement_list
END 
```

## IFNULL

`ifnull(exp1, exp2)`

exp1如果为空值，那么返回的结果为exp2的值；
exp1如果部位空，那么就返回exp1的值；


## WHILE语句

```
while 条件 do
内容
end while
```


## repeat语句

```
repeat 
    内容
    until 条件   
end repeat
```

## loop语句

```
loop名字： loop
    内容
if 条件 then
    leave loop名字; 
end if;
end loop;
```
